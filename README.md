# EquiPlanner

## Overview
EquiPlanner is a JavaFX desktop application for managing horses, riders, instructors, and lessons at an equestrian facility. It provides an intuitive interface for CRUD operations, lesson scheduling, and viewing combinations of riders and horses.

---

## Starting the Application
1. Clone this repository.  
2. Navigate to the `EquiPlanner` Java file inside `src`.  
3. Run the application from your IDE or command line.  

**Note:** When creating a lesson and adding a horse to a rider, you need to click on that row **three times** to register the selection.  

## Setting up the Database

1. Install XAMPP.
2. Open **phpMyAdmin**.
3. Create a new database called `equiplanner_db`.
4. Import the provided `equiplanner_db.sql` file that can be found in the database folder.
5. Start the JavaFX application.

---

## CRUD Functions
- Not all objects have full CRUD functionality yet due to time constraints.  
- The **Horse** model has 100% CRUD implemented.  
- The database comes with some initial data, but you can use the CRUD functionality inside the application to test and modify the data.
